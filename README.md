# 日語單字跟讀練習

把日語基礎單字依序一個接一個唸出來，每個字之間留空檔讓你跟著複誦。
純前端、單一 HTML 檔，使用瀏覽器內建的日語語音合成，免安裝、免伺服器。

## 線上使用

👉 **https://ccqqder.github.io/ja-voca/**

手機、電腦皆可，點開即用。

## 功能

- 依序自動朗讀（用「讀音 かな」發音，避免破音字唸錯）
- 可調語速、每字後停頓秒數（跟讀用）、每字重複次數
- 循環播放、可選是否補唸中文意思
- 依主題分組，點清單任一單字即可從該處開始
- 快捷鍵：空白鍵＝播放/暫停、←/→＝上一個/下一個

## 發音來源

使用 Windows / 瀏覽器內建語音合成（Web Speech API）。
若顯示「偵測不到日語語音」，請改用 **Microsoft Edge**，或於系統安裝日語語音套件。

## 內容

收錄日語基礎單字共 34 個，依主題分組，供個人與同學日常練習使用。
